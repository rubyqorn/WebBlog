# WebBlog

This is my own web application where I will write articles and news from IT. Here you can leave your comments participate in discussions with another users, read articles and news. This application based on PHP programming language, Laravel Framework, HTML/CSS, Bootstrap 4, MySQL and some Javascript.

# Technologies stack:

* [PHP](https://php.net)
* [MySQL](https://www.mysql.com)
* [Laravel](https://laravel.com)
* [HTML/CSS](https://www.w3.org/standards/webdesign/htmlcss)
* [Bootstrap](https://getbootstrap.com)
* [Javascript](https://javascript.com)
